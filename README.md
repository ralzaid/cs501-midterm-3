# Midterm Q3 – Lazy Lists

## Overview
This project creates a scrollable list of student names using Jetpack Compose.

The screen displays a list of 20 (21 with my name!!) student names using a LazyColumn. Each name is displayed inside a Card, and the list is vertically scrollable. A title ("Roster") is shown at the top, and consistent pink styling is used across the UI.

## AI Usage Disclosure
I used ChatGPT on web to ensure all requirements were met and maintain consistent styling with the previous questions and additional styling help.
